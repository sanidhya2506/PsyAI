# 🧠 PsyAI – A Calm Space for Your Mind

🌐 **Live Demo:** https://psyai-45nt.onrender.com  

---

## ✨ Overview

**PsyAI** is an AI-powered mental health companion designed to provide a safe, calm, and judgment-free space for users to express their thoughts and emotions.

It allows users to:
- Share how they feel 💭  
- Reflect on their emotions 🌱  
- Get supportive AI-driven responses 🤖  

This project focuses on **mental well-being, accessibility, and simplicity**, making emotional support more approachable through technology.

---

## 🚀 Features

- 💬 **Interactive AI Chatbot**  
- 🎨 **Calm & Minimal UI Design**  
- 📱 **Responsive (Mobile-Friendly)**  
- 🔐 **Secure API Handling using Environment Variables**  
- ⚡ **Fast Deployment using Render**

- Render (Hosting)  

---

## ⚙️ How It Works

1. User types a message in the chat interface  
2. Message is sent to the Flask backend (`/chat` endpoint)  
3. Backend processes the request using AI API  
4. AI generates a response  
5. Response is displayed in a clean chat UI
